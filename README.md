# 筋肉ログ

筋トレの重量・回数を記録するWebアプリです。

## 機能

- **種目記録** — 種目を選んで重量・回数を入力、ワンクリックで記録
- **前回参照** — 前回の重量・回数を自動表示、入力の手間を削減
- **自己ベスト更新演出** — 新記録達成時にフラッシュで通知
- **種目別グラフ** — 種目ごとの重量推移を折れ線グラフで可視化
- **自己ベスト一覧** — 各種目の最高重量を一覧表示
- **レベルシステム** — 累計ボリューム（重量×回数）でレベルアップ
- **連続日数カウント** — 何日連続でトレーニングしたか表示
- **データ永続化** — LocalStorageに自動保存

## 技術スタック

- **React 19**
- **TypeScript**
- **Vite**
- CSS（フレームワークなし）

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## ビルド

```bash
npm run build
```
